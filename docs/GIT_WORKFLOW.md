### Commit Message Format

<ul>
  <li>Adopt this format:</li>
  <ul>
    <li>[[Ticket-ID]] Short description</li>
  </ul>
  <li>Example:</li>
    <ul><li>[EP01-FT01-TK01] (task description)</li>
  </ul>
</ul>

### Merge Rules
<ul>
  <li>No direct push to main</li>
  <li>Feature branch → pull request → merge → Develop</li>
  <li>Develop → pull request → merge → Main</li>
  <li>At least one team member reviews</li>
</ul>
